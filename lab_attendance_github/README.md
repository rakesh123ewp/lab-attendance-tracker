# Lab Attendance Tracker
A Django-based Lab Attendance Management System for **East West Institute of Technology**.

## Features
- Teacher/Admin login
- Student self-login portal
- Lab, session & holiday management
- Attendance reports
- East West logo watermark

## Local Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
